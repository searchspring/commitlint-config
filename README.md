# commitlint-config
For clean commits!

```bash
npm i -D @searchspring/commitlint-config
```

Basically just adds the "wip" label.

The full set of commit labels is:

```json
[
    "build", 
    "ci", 
    "chore", 
    "docs", 
    "feat", 
    "fix", 
    "merge", 
    "perf", 
    "refactor", 
    "revert", 
    "style", 
    "test", 
    "wip"
]
```

Check out commitlint info here: https://github.com/conventional-changelog/commitlint
